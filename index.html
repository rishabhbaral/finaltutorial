Now that we know how many of our 1037 data points fall into each category, we can begin splitting the table into minitables, one for each period. This will make analysis infinitely easier and will allow us to see a comparison between teams in each period, separately.

To split the table, we will query the larger dataframe using the iloc command in order to retrieve just those rows from the table. In each row, we only need up to column 7 (index 6) for the analysis.

In [8]:
team_data_period_1 = NBA_Data.iloc[:151, :7]

Now the problem is that the data spans 8 years in the first bin. As such, teams are repeated. We don't want to have multiple points for the same team, so we need to find the Unique Team Names.

In order to do so, we will use the columnar "unique" function available for Pandas DataFrames.

In [9]:
teams_1 = team_data_period_1['Team'].unique()

Now we need to create a mini-array of the data we just extracted from the table. Let's use a loop over all the teams, adding the necessary info to a blank array.

The array we create here will come in handy later when we analyze the data and eventually plot it.

In [10]:
team_period_1 = []
for team in teams_1:
    team_period_1.append(team_data_period_1[team_data_period_1.Team == team])

Now that we have an array, we need to create a dataframe that we can use for plotting later.

In [11]:
team_data_1 = pd.concat(team_period_1)

Since there are roughly 8 years of data in each bin, we need to look at averages from the 8 year period. We will do this by calling the columnar "mean" function on the colums labeled Win % and Payroll.

In [14]:
mean_payroll_per_team_1 = []
mean_winning_pct_per_team_1 = []
for team in teams_1:
    mean_payroll_per_team_1.append(team_data_1.loc[team_data_1['Team'] == team, 'Payroll'].mean())
    mean_winning_pct_per_team_1.append(team_data_1.loc[team_data_1['Team'] == team, 'W/L%'].mean())

Now that we have calculated the averages for 'Payroll' and 'W/L%' over the first period, we can now make a smaller dataframe solely for this period in the data.

In [15]:
team_avgs_period_1 = pd.DataFrame() 
team_avgs_period_1['Team'] = teams_1
team_avgs_period_1['Mean Payroll'] = mean_payroll_per_team_1
team_avgs_period_1['Mean Winning Percentage'] = mean_winning_pct_per_team_1

And now that we have a dataframe containing the data we need from this time period, we can plot the data to see which teams were especially efficient.

In [17]:
period_1_NBA = team_avgs_period_1.plot(kind = 'scatter', title = "Payroll vs Win Percentage, 1984-1992", x = 'Mean Payroll', y = 'Mean Winning Percentage')
for idx, row in team_avgs_period_1.iterrows():
    period_1_NBA.annotate(row['Team'], (row['Mean Payroll'], row['Mean Winning Percentage']))
